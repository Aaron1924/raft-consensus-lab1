#INSTRUCTION

#1. Create a new repository on GitHub and clone it to your computer.

#2. install dependencies
```bash
pip install -r requirements.txt
```

#3. Run the server, every server is a terminal
```bash
py server.py {id}
or
python3 server.py {id}
```

#4. Run the client
```bash
py client.py
or
python3 client.py
```
